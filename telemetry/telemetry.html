Topic 1: Introduction

Welcome to your permanent dark mode interface. This application features a fixed top navbar and a secondary navigation panel pinned tightly to the left viewport frame. Click any menu link on your left to instantly jump down to that specific topic card.

Topic 2: Getting Started

The layout uses hardcoded CSS variables directly inside the root selector. Since the dark theme is now permanent, the template code is lighter, faster to load, and completely free of JavaScript event listeners.

Topic 3: Core Features

This layout uses CSS flexbox properties to cleanly separate block elements. The left menu utilizes a fixed height configuration combined with overflow-y: auto, ensuring that even if you have 50 topics in your sidebar list, it will scroll independently of the main content view window.

Topic 4: Advanced Styling

Notice the scroll-margin-top: 80px applied to our cards. This crucial line tells the web browser to leave exact padding space above the card when navigating with sidebar links. This stops the top fixed panel from accidentally resting on top of card headers.

Topic 5: Conclusion
